# 注释审阅清单（适用于后续变更）

- 行尾注释：
  - 使用中文，简洁描述该行代码功能。
  - 能对齐时尽量在同一列对齐，过长行不强制。
- 块注释：
  - 在 `{` 同行添加中文说明，`{` 后留一个空格。
  - 示例：`if (cond) { // 说明该块意图 }`。
- 复杂逻辑：
  - 逻辑前添加多行 `//` 中文说明，分点清晰。
- 函数注释：
  - 删除 Doxygen 的 `@brief/@param/@retval`。
  - 在函数名后的 `{` 同行添加中文简述。
- 大括号：
  - 左右对称，保持同一列对齐。
- 代码示例：
  - `HAL_Init();                      // 初始化HAL库`
  - `if (ok) { // 检查合法性 }`

审核通过标准：满足以上条目且不影响编译与格式化。
